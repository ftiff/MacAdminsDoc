# Tools for MacAdmins

## Editors
### TextEdit
Being free and available on every Mac, this is a no-brainer.

- Make sure you edit in text-only (CMD+SHIFT+T)
- Add it to your Dock for quick drag 'n drop
- in CLI, use `open -e [file]` to open in TextEdit
